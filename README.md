# CODSOFT-1
Task 1 is a number guessing game where the system generates a random number and the user guesses it with feedback, limited attempts, and score tracking.
Task 2 is a student grade calculator that takes subject marks as input, computes total and average, assigns grades, and displays pass or fail results.
Task 3 is an ATM interface that allows users to deposit, withdraw, and check balance using a menu-driven system with proper input validation.
Task 5 is a student management system that allows users to add, edit, search, delete, display, and store student records using a menu-driven interface with input validation.
